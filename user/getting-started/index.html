クイックスタート¶

1. 起動¶

  1. 配布物を展開します。
  2. 実行ファイル（iRIC_Input_Checker_vX.Y.Z.exe）を起動します。

配布物をまだ入手していない場合は、インストール を参照してください。

2. 最小手順（確認モード）¶

  1. 実行モードで 確認（1ケース） を選択します。
  2. 基準ケース に .ipro または project.xml を指定します。
  3. 保存先フォルダ を指定します。
  4. 簡易レポート（HTML形式） と 簡易レポート（PDF形式） を有効化します。
  5. 実行 を押します。

3. 最小手順（比較モード）¶

  1. 実行モードで 比較（2ケース） を選択します。
  2. 基準ケース と 比較ケース を指定します。
  3. 保存先フォルダ を指定します。
  4. レポート出力設定を確認し、実行 を押します。

4. 出力確認¶

  • 実行完了ダイアログで レポートを開く または 出力フォルダを開く を選択します。

補足¶

  • project.xml を指定する場合は、同じフォルダ内に必要なプロジェクトファイル一式が存在することを確認してください。
