確認モード（Check）¶

概要¶

確認モードは、単一ケースの入力条件を読み取り、実行前確認のためのレポートを生成する機能です。

入力項目¶

  • 基準ケース: .ipro または project.xml
  • 保存先フォルダ: 出力先フォルダ（入力プロジェクト配下は指定不可）

出力内容¶

  • 簡易レポート（HTML形式）
  • 簡易レポート（PDF形式）
  • 詳細レポート（マークダウン形式、任意）
  • JSON / 格子属性分布図 / Functionalグラフ（自動出力）

補足:

  • HTML を有効にすると web/report/index.html と web/detail/index.html が同時生成されます。
  • 詳細確認は簡易レポート側の 詳細確認を開く から遷移します。

詳細設定¶

  • ログレベル（通常: INFO）
  • 格子属性の分布図 最大件数
  • Functional グラフ 最大件数

注意事項¶

  • project.xml 指定時は、同一フォルダ内にプロジェクト構成ファイルがそろっている必要があります。
